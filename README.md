# SketchParty 🎨

Multiplayer drawing game with hand gesture control.

## Features
- Real-time multiplayer drawing (up to 4 players)
- Hand gesture detection with camera
- Turn-based game with word guessing
- Live scoring system
- 6 brush effects (Glow, Blur, Rainbow, Particle, Trail, Normal)

## Tech Stack
- Node.js + Express (Backend)
- Socket.io (Real-time sync)
- MediaPipe Hands (Hand detection)
- Canvas API (Drawing)
- Deployed on Railway

## Play Now
[https://sketch-party-production.up.railway.app](https://sketch-party-production.up.railway.app)

## How to Play
1. Open the URL
2. Enter your name
3. Create or join a room
4. Draw with hand or mouse
5. Guess others' drawings to earn points

## Local Development
```bash
npm install
npm start
```

Open `http://localhost:8000`
